---
title: "replace / replaceOrAppend — swap or add items"
description: "Return a new array with the first matching item replaced, or optionally append the new item when no match is found, without mutating the original."
---

`replace` and `replaceOrAppend` both return a new array with the first element that satisfies a `match` predicate swapped out for `newItem`. They differ in what happens when no match is found: `replace` returns a copy of the original list unchanged, while `replaceOrAppend` appends `newItem` to the end.

```typescript
function replace<T>(
  list: readonly T[],
  newItem: T,
  match: (item: T, idx: number) => boolean
): T[]

function replaceOrAppend<T>(
  list: readonly T[],
  newItem: T,
  match: (a: T, idx: number) => boolean
): T[]
```

<ParamField path="list" type="readonly T[]" required>
  The source array. Neither function mutates this array.
</ParamField>

<ParamField path="newItem" type="T" required>
  The replacement value inserted at the position of the first match.
</ParamField>

<ParamField path="match" type="(item: T, idx: number) => boolean" required>
  A predicate receiving the current element and its index. The first element for which this returns `true` is replaced.
</ParamField>

Both functions return a `T[]`. `replace` returns an unchanged copy when no match is found. `replaceOrAppend` appends `newItem` to the end when no match is found.

```typescript
import { replace, replaceOrAppend } from 'radash'

// Replace a primitive value
replace([1, 2, 3, 4], 20, n => n === 2)
// => [1, 20, 3, 4]

// Replace an object by id
type Task = { id: number; done: boolean }
const tasks: Task[] = [
  { id: 1, done: false },
  { id: 2, done: false },
  { id: 3, done: false },
]

replace(tasks, { id: 2, done: true }, t => t.id === 2)
// => [
//   { id: 1, done: false },
//   { id: 2, done: true },
//   { id: 3, done: false },
// ]

// No match — replace returns the original list unchanged
replace(tasks, { id: 99, done: true }, t => t.id === 99)
// => (same as tasks)

// No match — replaceOrAppend adds the item
replaceOrAppend(tasks, { id: 99, done: true }, t => t.id === 99)
// => [...tasks, { id: 99, done: true }]

// Use index in the predicate
replace(['a', 'b', 'c'], 'Z', (_item, idx) => idx === 1)
// => ['a', 'Z', 'c']
```
