---
title: "boil — reduce array to a single winner"
description: "Iteratively compare every element in an array using a custom function and return the single surviving element, or null for an empty array."
---

`boil` walks through an array pairwise, handing each pair to your `compareFunc` and keeping whichever value the function returns. This makes it the primitive behind custom max, min, or any other winner-takes-all reduction — without forcing you to extract a numeric key first.

```typescript
function boil<T>(
  array: readonly T[],
  compareFunc: (a: T, b: T) => T
): T | null
```

<ParamField path="array" type="readonly T[]" required>
  The source array to reduce. Returns `null` when the array is empty or nullish.
</ParamField>

<ParamField path="compareFunc" type="(a: T, b: T) => T" required>
  A function that receives the current winner and the next candidate and returns whichever should survive.
</ParamField>

Returns the single winning element of type `T`, or `null` if the array is empty.

```typescript
import { boil } from 'radash'

// Numeric max
const numbers = [1, 5, 3, 9, 2]
const greatest = boil(numbers, (a, b) => (a > b ? a : b))
// => 9

// Numeric min
const smallest = boil(numbers, (a, b) => (a < b ? a : b))
// => 1

// Object comparison
type Player = { name: string; score: number }
const players: Player[] = [
  { name: 'Alice', score: 82 },
  { name: 'Bob', score: 95 },
  { name: 'Carol', score: 74 },
]

const topPlayer = boil(players, (a, b) => (a.score > b.score ? a : b))
// => { name: 'Bob', score: 95 }

// Empty array
const result = boil([], (a, b) => a)
// => null
```
